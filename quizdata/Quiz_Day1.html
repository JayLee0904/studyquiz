Day1 (Page 60)

JP : アクリル板を間に挟んで仕事をする人たち。
KR : 아크릴판을 사이에 끼어서 일하는 사람들

JP : 今日のうちにこの仕事を仕上げなくてはいけないです。
KR : 오늘중으로 이일을 마무리하지 않으면 안됩니다.

JP : 物価は上がるのに給料は上がらないので生活が大変です。
KR : 물가는 오르는데 월급은 오르지 않아 생활이 힘듭니다.

JP : 人々の話題に上っている。
KR : 사람들에게 화제에 오르고 있어요.

JP : 空き缶や空き瓶はリサイクルしましょう。
KR : 빈 캔이나 빈병은 재활용 합시다

JP : 今期は頑張りましたが、空振りに終わりました。
KR : 이번 분기 열심히 했지만 헛스윙(헛수고)로 끝났습니다.

JP : 彼の考え方は明らかに間違っている。
KR : 그의 사고방식은 분명히 틀렸다.

JP : 彼女の料理の腕は、確かだ。
KR : 그녀의 요리솜씨는 확실하다.

JP : うちの家は窓を開けたらら爽やかな風が部屋に入ってきます。
KR : 우리집은 창문을 열면 상쾌한 바람이 방에 들어옵니다.

JP : 手帳のようにパカっと開くタイプのスマートフォンのカバー
KR : 수첩처럼 탁 열리는 타입의 스마트폰 커버

JP : 毎日疲れきって会社から帰って来る。
KR : 매일 완전히 지친 몸으로 회사에서 돌아온다.

JP : 彼はかなり色あせた服を着ている。
KR : 그는 상당히 색 바랜 옷을 입고 있다.

JP : もう彼への愛は、冷めてしまった。
KR : 이미 그에 대한 사랑은 식어 버렸다.

JP : 線路の向うに、ひまわりがたくさん咲いている。
KR : 선로 저편에 해바라기가 많이 피어있다.

JP : 刑事は、あたかも私が犯人であるかのように睨んだ。
KR : 형사는 마치 내가 범인인 것처럼 째려보았다.

JP : 砂糖と塩を間違えるなんて、いかにも君らしいよ。
KR : 설탕과 소금을 헷갈리다니 과연 너답다.

JP : このへんで、今日の仕事は終わりにしましょう。
KR : 이쯤에서 오늘 일은 끝내기로 합시다.

JP : やっぱり、彼女の予想が当たって、雨になったね。
KR : 역시 그녀의 예상이 맞아서 비가 오는군.

JP : この先の交差点で、トラックとバイクがぶつかった。
KR : 이 앞의 교차점에서 트럭과 오토바이가 부딫혔다

JP : 彼女は、いやなことはすぐ忘れる、あっさりした性格です。
KR : 그녀는 좋지 않은 일은 바로 잊어버리는 뒤끝없는 성격입니다.

JP : あんな男とは、きれいさっぱり別れました。
KR : 저런 남자와는 깔끔하게 헤어졌습니다.

JP : 梅雨が終わって、空がからっとと晴れ上がった。
KR : 장마가 끝나고 하늘이 활짝 갰다.

JP : 考えがまとまらないので困っています。
KR : 생각이 정리되지 않아서 어려움을 겪고 있습니다.

JP : 彼は彼女に心を寄せている。
KR : 그는 그녀에게 마음을 두고 있다.

JP : 私の部屋は、あとで片付けますから、そのままにしておいてください。
KR : 제 방은 나중에 치울테니 그대로 놔두십시오.

JP : この件につきましては、後ほど詳しくご説明いたします。
KR : 이 건에 관해서는 나중에 자세히 설명드리겠습니다.

JP : 彼女の顔は、自信にあふれていた。
KR : 그녀의 얼굴은 자신감이 넘쳐 있다.

JP : 車の後部座席にこぼれたジュース
KR : 차 뒷좌석에 엎질러진 주스

JP : この頃、仕事が忙しくて、あまりゲームはやってない。
KR : 최근 일이 바빠서 그닥 게임은 하지 않네요.

JP : 彼はたいしてえらくもないのに、いつもいばっている。
KR : 그는 그다지 대단하지도 않은데 항상 우쭐댄다
